Console App : Bloomify (C#)

Bloomify Console App is a console-based productivity tool that helps users improve focus and develop healthy work habits. It combines the Pomodoro Technique with interactive blooming flower animations to make time management engaging and motivating. Users can set custom focus and break timers, track sessions in a history log, and visualize progress through blooming flowers. Built using structured programming principles, the app demonstrates practical application of variables, loops, control structures, arrays, and custom methods in a console environment.
